# 🎬 Sistema de Cinema em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## 📌 Funcionalidades

- Exibir cadeiras disponíveis e ocupadas
- Reservar cadeira
- Cancelar reserva
- Listar cadeiras livres
- Mostrar resumo (total, livres e ocupadas)

## 🧠 Conceitos aplicados

- Estruturas de repetição
- Condicionais
- Listas
- Funções
- Validação de dados
- Menu interativo

## 🚀 Objetivo

Desenvolver um sistema simples de gerenciamento de cadeiras para consolidar conhecimentos em lógica de programação.

## 💻 Como executar

Execute o arquivo principal com Python:

```bash
python main.py
